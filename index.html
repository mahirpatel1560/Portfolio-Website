Raspberry Pi Home Automation

The project showcases a comprehensive home automation system built using a Raspberry Pi. This system integrates smart devices such as lights, thermostats, security cameras, voice assistants, outlet plugs, and smart TVs, allowing centralized control through a user-friendly interface. By leveraging Python and various IoT protocols, the project includes automation scripts and voice command functionalities, enhancing convenience and efficiency in managing home environments.
